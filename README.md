# ⚡ OpsAgent — AI-Native DevOps Copilot

Built by **Arush Kumar** & **Ayushi Shukla**

## Features
- 🎙️ Meeting Analyzer
- ⚡ Sprint Planner
- 🔍 PR Reviewer
- 🚨 Incident Explainer
- 💻 Natural Language DevOps Commands

## Stack
- Next.js + TypeScript + TailwindCSS
- Groq API (LLaMA3) — Free tier
- Deployed on Vercel

## Setup
```bash
git clone https://github.com/arushkumar-aiml/OpsAgent
cd OpsAgent/frontend
npm install
# Add GROQ_API_KEY to .env.local
npm run dev
```
